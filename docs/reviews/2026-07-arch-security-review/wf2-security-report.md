### [MEDIUM] In-memory decrypt accumulates plaintext with read_to_end, leaking un-zeroized copies through Vec reallocation

- **Defect:** decrypt_with_helper() decrypts into `let mut plaintext = Vec::new(); decryptor.read_to_end(&mut plaintext)`. read_to_end grows the Vec by repeated doubling; each reallocation copies existing plaintext into a new allocation and frees the old backing buffer WITHOUT zeroizing it.
