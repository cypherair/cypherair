# CypherAir Agent Guide

CypherAir is an offline OpenPGP encryption app for Apple platforms.

`CLAUDE.md` at the repository root is the single agent-facing source of truth —
project constraints, security boundaries, build commands, testing, releases, and
workflow rules all live there. Read it before making any change; this file
intentionally contains no rules of its own and duplicates nothing. The deeper
canonical documents under `docs/` are indexed from `CLAUDE.md`.
