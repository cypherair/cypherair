# Cypher Air

Offline OpenPGP encryption for everyone.

---

**Cypher Air** is an iOS app that lets ordinary people encrypt messages and files — completely offline, with no servers, no accounts, and no tracking.

It uses modern OpenPGP standards (Ed25519, X25519, AES-256-OCB) and is fully compatible with GnuPG. Your private keys stay in the iOS Keychain, protected by Face ID.

### Key Principles

- **Truly offline** — zero network access, by design
- **Standard-compliant** — encrypted output works with any OpenPGP tool
- **Built for humans** — no cryptography knowledge required
